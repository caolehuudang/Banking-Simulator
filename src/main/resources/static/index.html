# Endpoint Method Function
1 /getAllBanks GET View all bank in the system
2 /addNewBank POST Add new Bank into the system
3 /addNewAccount POST Add new user into the bank
4 /getAllAccounts GET View all users in the bank
5 /getAccountBySerial POST View user by serial number
6 /depositMoney POST Deposit money into the bank
7 /withDrawMoney POST Withdraw money from the bank
8 /verifyOTP/{id}/{otp} GET Verify OTP in email
